#  Habit Tracker App

A full-stack **Habit Tracker Application** built with modern web technologies to help users build consistency, track progress, and analyze habits effectively.

---

##  Overview

This application allows users to **create, manage, and track daily habits** with powerful analytics such as weekly completion, monthly activity, and progress ratios.

It is designed with a clean UI and scalable architecture using a monorepo structure.

---

##  Features

* Add, Edit, Delete Habits
*  Daily Habit Tracking
*  Weekly Completion Insights
*  Daily Progress Analysis
*  Monthly Activity Overview
*  Completion Ratio Metrics
*  Modern UI using ShadCN

---

##  Installation & Setup

### 1️ Clone Repository

```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
npm run dev
```

---

###  Frontend Setup

```
cd frontend
npm install
npm run dev
```

---

### 4️⃣ Admin Panel Setup

```
cd admin
npm install
npm run dev
```

---

##  Environment Variables

Create a `.env` file inside the **backend** folder:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
```
